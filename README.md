# Handwritten Digit Recognizer using CNN

A deep learning project that recognizes handwritten digits (0–9) using Convolutional Neural Networks (CNN) and the MNIST dataset.

## Features
- CNN-based digit classification
- Trained on MNIST dataset
- Achieved high accuracy
- Visual prediction output

## Technologies Used
- Python
- TensorFlow/Keras
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
MNIST Handwritten Digits Dataset

## Accuracy
Achieved approximately 98% test accuracy.
